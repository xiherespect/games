# 俄罗斯方块产品规格

## 核心玩法

经典的俄罗斯方块游戏：从顶部随机掉落不同形状的方块，玩家通过移动和旋转方块，将其放置在底部，填满一行即可消除并获得分数。

## 基本功能

### 游戏控制

| 按键/操作 | 功能 |
|----------|------|
| ← / A | 向左移动 |
| → / D | 向右移动 |
| ↓ / S | 加速下落 |
| ↑ / W | 旋转方块 |
| 空格键 | 直接落到底部 |
| P | 暂停/继续 |

### 游戏机制

- **方块类型**: I, O, T, S, Z, J, L 7种经典方块
- **难度级别**: 3个难度（简单/中等/困难）
  - 简单：方块下落速度慢
  - 中等：方块下落速度中等
  - 困难：方块下落速度快
- **得分规则**:
  - 消除1行：100分 × 当前等级
  - 消除2行：300分 × 当前等级
  - 消除3行：500分 × 当前等级
  - 消除4行：800分 × 当前等级
- **游戏结束**: 当新方块无法放置时游戏结束

### UI界面

- 游戏画布：10列 × 20行网格
- 下一个方块预览区域
- 当前分数显示
- 最高分记录（本地存储）
- 难度选择
- 开始/重新开始按钮
